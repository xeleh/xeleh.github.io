Herramientas Unity

Editor Enhancer

2020 -

Paquete de utilidades destinadas a mejorar el editor de Unity: modo oscuro, barra lateral, etc.

GameFlow

2013 - 2019

Extensión de visual scripting diseñada para simplificar la programación de videojuegos.

Instancer

2016

Herramienta para generación y colocación de instancias a lo largo de guías extraídas de archivos Autocad.

Soccer Data Editor

2016

Editor visual de conjuntos de datos 3D obtenidos mediante escaneo en tiempo real de partidos de fútbol.

Course Data Editor

2016

Editor visual de conjuntos de datos GPS de carreras de Fórmula E.

XT Utilities

2016

Paquete de pequeñas utilidades para el editor de Unity.

XT Reorder

2015

Extensión que habilita la posibilidad de reordenar componentes en la ventana Inspector mediante drag and drop.

Adaptación de Vitrun para jugar con el controlador Leap Motion. Unity.

Vitrun

El clásico juego de rodar la bola, pero con un toque muy personal. Unity.

Serie de juegos de casino y apuestas para dispositivos móviles. J2ME.

Fantastic League

2005-

Plataforma de apuestas sobre eventos deportivos virtuales. C++.

Gran Doradium

2003 - 2004

El primer juego para una nueva generación de máquinas recreativas tipo B. Java y OpenGL.

Quadryx

2002

Versión para ordenador de un juego de tablero de origen medieval. Director.

V3ga

2000 - 2001

Juego de naves al estilo de R-Type para jugar en el navegador. Java.
